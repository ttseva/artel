# Артель
> В рамках курсовой работы РТУ МИРЭА 3 курс по дисциплине "Разработка клиент-серверных приложений"
## Платформа для локальных мастеров

Артель — это объединение мастеров или творческих людей, которые работают рядом, каждый над своим делом, без общей финансовой ответственности, но с общим пространством и духом взаимопомощи.
