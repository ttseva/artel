# Artel Frontend

Клиентская часть платформы для локальных мастеров «Артель».

## Стек технологий
- React
- Vite
- React Router
- Axios
- CSS

## Установка и запуск

1. Перейдите в папку клиента:
   ```bash
   cd client
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите клиент:
   ```bash
   npm run dev
   ```

4. Откройте приложение в браузере:
   ```text
   http://localhost:5173
   ```

## Доступные скрипты
- `npm run dev` — запуск клиента в режиме разработки
- `npm run build` — сборка production-версии
- `npm run preview` — локальный просмотр production-сборки
- `npm run lint` — проверка кода ESLint

## Структура
- `/src/api` — функции для запросов к REST API
- `/src/components` — переиспользуемые компоненты интерфейса
- `/src/context` — глобальное состояние авторизации
- `/src/pages` — страницы приложения
- `/src/components/app` — основной компонент приложения и маршрутизация

## Основные страницы
- `/` — каталог изделий
- `/auth` — вход и регистрация
- `/user/me` — профиль пользователя (только авторизованным)
